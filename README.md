# MediMinderApp

## Project Overview
MediMinderApp is a medication reminder application designed to help users manage their medication schedules effectively. The app sends reminders to users to take their medications on time, ensuring adherence to prescribed therapies. With a user-friendly interface and customizable settings, MediMinderApp offers a reliable solution for anyone looking to maintain their medication routine.

## Features
- **Medication Scheduling**: Users can set up schedules for multiple medications with specific time intervals.
- **Notifications**: Reminders are sent to the user's device at the scheduled times to prompt medication intake.
- **User-friendly Interface**: The app features an intuitive design that makes it easy for users to manage their medications.
- **Customization Options**: Users can customize reminder times, recurring schedules, and more.
- **Data Security**: Sensitive user data is stored securely, ensuring privacy and protection.

## Installation Instructions
1. **Clone the Repository**: 
   ```bash
   git clone https://github.com/2FAST4Uh/MediMinderApp.git
   ```
2. **Navigate to the Project Directory**:
   ```bash
   cd MediMinderApp
   ```
3. **Install Dependencies**:
   ```bash
   flutter pub get
   ```
4. **Run the Application**: 
   ```bash
   flutter run
   ```

## Usage Information
- Upon launching the app, users will be greeted with a welcome screen.
- To add a medication, click on the "Add Medication" button and fill in the required details, including medication name, dosage, and schedule.
- Users can view their medication list and edit or delete entries as needed.
- Ensure notifications are enabled to receive timely reminders for taking medications.
